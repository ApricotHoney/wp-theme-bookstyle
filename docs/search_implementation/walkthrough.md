# 検索機能実装 Walkthrough

## 実施した変更
1. **カスタムタクソノミー `color` の追加** (`functions.php`)
    - 投稿に「カラー」属性を追加しました。管理画面から色を選択・追加できます。

2. **検索フォームの刷新** (`header.php`)
    - テキスト検索ボックスを削除し、以下のフィルタリングUIに変更しました。
        - **カテゴリ**: ドロップダウン（「ジャンルでさがす」）
        - **並び替え**: ドロップダウン（新しい順、古い順、ランダム）
        - **色検索**: カラーパレット（黒、青、オレンジ、緑、白、赤）

3. **検索ロジックの実装** (`front-page.php`)
    - `WP_Query` の引数を修正し、GETパラメータ (`cat`, `sort`, `color`) を受け取ってクエリを変更するようにしました。

4. **スタイリング** (`style.css`)
    - カラー検索用の丸いアイコンスタイルと、ドロップダウンのスタイルを追加しました。

## 確認方法

### 管理画面での設定
1. ダッシュボード > 投稿 > カラー (Colors) から、必要な色（Black, Blue, Orange, Green, White, Red）が登録されているか確認してください（未登録の場合は登録してください）。
2. 各投稿の編集画面で、「カラー」を適切に設定してください。

### フロントエンドでの動作確認
1. **カテゴリ検索**: 「ジャンルでさがす」からカテゴリを選び、自動的に（または検索ボタンで）絞り込まれることを確認。
2. **並び替え**: 「並び替え」から「古いもの順」などを選び、表示順が変わることを確認。
3. **色検索**: 任意の色の丸アイコンをクリックし、その色が設定された投稿のみが表示されることを確認。
    - 選択中の色は少し大きく、または枠線が表示されます。
