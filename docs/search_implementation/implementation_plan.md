# 実装計画: 検索・並び替え・色検索機能

## 概要
WordPressテーマ `bookstyle_v4` に、カテゴリ検索、並び替え、色検索機能を追加します。
ユーザーの要望に従い、テキスト検索ボックスは除外し、カテゴリ・並び替え・色の選択のみでフィルタリングできるようにします。

## 対象ファイル
- `functions.php`: カスタムタクソノミー `color` の追加、クエリパラメータの処理（必要であれば）
- `header.php`: サイドバー（ナビゲーション）内に検索フォームを追加
- `front-page.php`: `WP_Query` の引数をGETパラメータに応じて動的に変更するロジックを追加
- `style.css`: 検索フォーム、カラースウォッチのスタイル定義

## 詳細な変更内容

### 1. カスタムタクソノミーの追加 (`functions.php`)
- `color` タクソノミーを `post` タイプに追加します。（UIから色を選択して投稿に紐付けられるようにする）
- 色のターム（例: Red, Blue, etc.）は管理画面から登録することを想定しますが、初期データとしていくつかの色が必要かもしれません。
- 登録する色: Black, Blue, Green, Orange, Red, White, etc.

### 2. 検索フォームの実装 (`header.php`)
- サイドバーのナビゲーション下部（`nav`タグの後など）に検索フォーム配置。
- **フォーム要素**:
    - **カテゴリ選択**: `wp_dropdown_categories` を使用、または手動で `<select>` を生成。「ジャンルでさがす」
    - **並び替え**: `<select name="sort">`。「並び替え」（新しい順、古い順、ランダム）
    - **色選択**: ラジオボタン（`<input type="radio" name="color">`）を使用し、CSSで色の丸アイコンとして表示。「色でさがす」
    - **送信ボタン**: フォームの変更で自動送信するか、ボタンを設置。今回は自動送信または「検索」ボタンアイコンでの送信を想定（JSで実装可）。ユーザー要望の「テキストボックス不要」に従い、`input[type="text"]`は削除。

### 3. クエリの変更 (`front-page.php`)
- デフォルトの `new WP_Query` に引数を追加。
    - `cat`: カテゴリID (`$_GET['cat']`)
    - `orderby` & `order`: `$_GET['sort']` に応じて分岐 ('date' DESC, 'date' ASC, 'rand')
    - `tax_query`: `color` タクソノミーによる絞り込み (`$_GET['color']`)

### 4. スタイリング (`style.css`)
- ドロップダウンのスタイル調整
- 色選択ラジオボタンを円形のカラーパレットにするスタイル

## 検証計画

1. **管理画面**:
    - [ ] `color` カスタムタクソノミーが表示され、投稿に色を設定できること。
    - [ ] テスト投稿に各色、各カテゴリを設定する。
2. **フロントエンド表示**:
    - [ ] サイドバーに検索フォームが表示されていること。
    - [ ] カテゴリ「すべて」から特定カテゴリを選んで検索→該当カテゴリのみ表示される。
    - [ ] 並び替えを変更→記事の順序が変わる（新しい順、古い順）。ランダムはリロードで変わる。
    - [ ] 色を選択→その色が設定された記事のみ表示される。
    - [ ] 複数の組み合わせ（カテゴリ＋色）で絞り込めること。
